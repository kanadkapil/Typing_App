## Features üöÄ

- **8 Premium Themes**: Choose from Carbon, Neon Nights, Cyberpunk, Neo Brutalist, and more.
- **Performance Analytics**: Real-time WPM and accuracy tracking with a detailed results graph (powered by Recharts).
- **Flexible Test Modes**: Support for both **Time** (15, 30, 60, 120s) and **Word** (10, 25, 50, 100 words) limits.
- **Mechanical Sound Effects**: Toggle-able typing sounds for a satisfying experience.
- **Local Persistence**: Saves your settings and typing history to your browser's local storage.
- **Keyboard Shortcuts**:
  - `Tab` - Restart test
  - `Esc` - Cancel current test and return to menu
- **Modern Tech Stack**: Built with **Vite**, **Taiwind CSS v4**, and **Lucide Icons**.

## Getting Started üõ†Ô∏è

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/typingapp.git
   cd typingapp
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
